# TalismanReward

Talisman Reward mod for Monster Hunter Rise/Sunbreak in Steam

## Installation

This talisman_rewards.lua and Talisman Reward folder place to reframework -> autrun folder in the Monster Hunter Rise.

```bash
MonsterHunterRise
└─reframework
  └─autorun
    ├─Talisman Reward
    | └─modUtils.lua
    └─talisman_rewards.lua
```
